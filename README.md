# vue3-todo-mvc

vue3+ts+scss+vite 实现的 `TodoMVC`,组件拆分比较严重,是为了更好的学习 vue3.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
